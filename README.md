# Data-Solve: Advanced Analytics & Query Challenges 🚀

### Solutions for StrataScratch SQL & Python Problems

This repository serves as a centralized hub for my solutions to technical challenges from **StrataScratch**. It demonstrates my ability to solve complex data engineering and data science problems using multiple frameworks, including **SQL**, **Python (Pandas)**, and **Polars**.

---

## 📌 Project Overview
In data roles, the tool you choose matters as much as the logic you apply. This project showcases how to tackle the same business logic across different environments, focusing on:

*   **Query Optimization:** Writing efficient SQL for high-performance data retrieval.
*   **Vectorized Operations:** Leveraging Pandas and Polars for fast, memory-efficient data manipulation.
*   **Logic Parity:** Implementing identical solutions across different syntax styles (e.g., CTEs in SQL vs. Method Chaining in Polars).

## 🛠️ Tech Stack
*   **SQL:** PostgreSQL, MySQL (Window Functions, CTEs, Complex Joins).
*   **Python:**
    *   **Pandas:** Data cleaning, aggregation, and transformation.
    *   **Polars:** High-performance DataFrame operations utilizing lazy evaluation and concurrency.
*   **Platform:** Problems sourced from [StrataScratch](https://stratascratch.com).
